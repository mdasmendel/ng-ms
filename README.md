# Start project

### Clone the project to your local machine

```bash
$ git clone [repository url]
```
### Start server

#### Install dependencies

```bash
$ cd backend
$ npm i
```

#### Run in terminal:
```bash
$ npm start
```

### Start client

#### Install dependencies

```bash
$ cd messages-app
$ npm i
```

#### Run in terminal:
```bash
$ npm start
```
